What is a Joy-Con?

A Joy-Con is a very useful controller designed for the Nintendo Switch. It has the same features as you would see on a "normal" game controller, but it also has more on top of that. You can see some of the functions in the list bellow. Note that the list specifies the functions of each individual controller unless otherwise specified.

  • 11 different buttons that can be used for different things.
  • An anologue joystick that also works as one of the buttons.
  • A special connector for connecting the Joy-Con to a charger or to control things by wired communication.
  • Accelerometer and gyroscope to control things by moving the controller around.
  • NFC-reader (right only).
  • Infrared camera (right only).

The Joy-Con was originally designed for the Nintendo Switch but can be used by other devices if you install external software. For Windows you can for example use BetterJoy. If you use MAC or Linux you can probably find similar software somewhere else. For example on GitHub. Note that such software is installed on your own risk and may not work on all devices.
